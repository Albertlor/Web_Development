热学的"主角"其实是气体

例如飞机引擎使用的otto cycle

其工作原理就是靠气体对外界做功完成的

ΔU=ΔQ+ΔW

1 => 2: 绝热收缩过程，气体对外界做负功先对燃油进行预热

2 => 3: 等容过程，急剧升高温度引爆燃油

3 => 4: 绝热膨胀过程，气体对外界做正功提供飞机驱动力

4 => 1: 等容过程，将燃油降温再排出
